# Environment Variables (Template)

## Gemini (Google AI)
- GEMINI_API_KEY

## Cloudinary
- NEXT_PUBLIC_CLOUDINARY_CLOUD_NAME
- CLOUDINARY_API_KEY
- CLOUDINARY_API_SECRET

## Supabase
- NEXT_PUBLIC_SUPABASE_URL
- NEXT_PUBLIC_SUPABASE_ANON_KEY
- SUPABASE_SERVICE_ROLE_KEY

## Krea.ai (optional legacy)
- KREA_API_TOKEN
- KREA_MODEL_ENDPOINT
- IMAGE_WIDTH
- IMAGE_HEIGHT
- IMAGE_STEPS

Notes:
- Use `.env.example` as the authoritative key list.
- Never commit `.env.local`.
